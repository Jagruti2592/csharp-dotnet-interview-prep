# C# / .NET Interview Preparation Projects

This repository contains small focused projects built to master important C# concepts commonly asked in .NET developer interviews.

Each project focuses on a single concept and demonstrates its practical usage through a mini application.

## Topics Covered

1. Generics
2. Delegates
3. Lambda Expressions
4. Events
5. Extension Methods
6. LINQ
7. Nullable Types
8. Dynamic Types
9. Exception Handling
10. Asynchronous Programming (Async/Await)

## Goal

The goal of this repository is to build a strong understanding of C# fundamentals by implementing small projects and pushing them to GitHub daily.

Each project includes:

* Explanation of the concept
* Practical example
* Clean code implementation

## Tech Stack

* C#
* .NET
* Console Applications
* LINQ
* Async Programming
